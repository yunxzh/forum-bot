Forum-Bot 是一个通用的论坛自动化机器人平台，支持自动签到、AI智能回复等功能。通过Web管理界面，可以轻松管理多个论坛站点。

## ✨ 核心特性

### 🌐 多站点支持
- 支持同时管理多个论坛站点
- 内置 DeepFlood、NodeLoc、LinuxDo 等预设模板
- 自定义 CSS 选择器，适配任何论坛架构

### 🔐 灵活的认证方式
- Cookie 登录（推荐）
- 账号密码自动登录

### 🤖 AI 智能回复
- 集成 OpenAI 兼容 API
- 智能内容分析，过滤垃圾帖子
- 自然语言回复生成
- 可配置回复长度和频率

### 📅 定时任务调度
- Cron 表达式配置执行时间
- 支持签到、回复、回帖任务
- 实时任务日志记录

### 🔔 多渠道通知
- Telegram
- 企业微信
- PushPlus
- 钉钉
- 飞书
- Bark
- SMTP 邮件
- Gotify

### 🎨 现代化 Web 管理界面
- 直观的仪表板
- 站点管理
- 任务日志查看
- 全局配置管理
